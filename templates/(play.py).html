/projeto

O script main.py é um programa que oferece uma interface de linha de comando para interagir com a classe CadastroUsuario, permitindo ao usuário adicionar, consultar, listar e remover usuários de um banco de dados.

1. Importação do Módulo


                    from utils.utilits import CadastroUsuario

  • from utils.utilits import CadastroUsuario: Importa a classe CadastroUsuario do módulo utilits, que está localizado no pacote utils. Isso permite que você utilize a classe CadastroUsuario no seu script principal.

    •

    2. Definição da Função main

    
                def main():
                    print('BEM VINDO AO CADASTRO DE USÚARIOS!')
                    opcao = input('Iniciar Aplicativo?? [s][n]: ')
                    
                    if opcao.lower() == 's':
                        print('-------------------------')
                        print('Iniciando o aplicativo...')
                        print('-------------------------')
                        cadastro = CadastroUsuario()

  • def main():: Define a função principal main, que é o ponto de entrada do programa.
  • print('BEM VINDO AO CADASTRO DE USÚARIOS!'): Exibe uma mensagem de boas-vindas.
  • opcao = input('Iniciar Aplicativo?? [s][n]: '): Solicita ao usuário que escolha se deseja iniciar o aplicativo ou não.
  • if opcao.lower() == 's':: Verifica se a opção fornecida pelo usuário é 's' (sim). O método .lower() garante que a comparação não seja sensível a maiúsculas/minúsculas.
  • print('Iniciando o aplicativo...'): Exibe uma mensagem indicando que o aplicativo está sendo iniciado.
  • cadastro = CadastroUsuario(): Cria uma instância da classe CadastroUsuario, o que permite usar os métodos para gerenciar usuários.

    3. Menu de Opções

    
                while True:
                    print("\nMenu:")
                    print('-------------------------')
                    print("1. Adicionar usuário")
                    print("2. Retornar usuário")
                    print("3. Listar todos os usuários")
                    print("4. Remover usuário")
                    print("5. Sair")
        
                    escolha = input("Escolha uma opção: ")

  • while True:: Inicia um loop que continuará executando até que o usuário escolha sair.
  • print("\nMenu:"): Exibe o cabeçalho do menu.
  • print("1. Adicionar usuário"): Exibe a opção para adicionar um usuário.
  • print("2. Retornar usuário"): Exibe a opção para retornar informações de um usuário com base no email.
  • print("3. Listar todos os usuários"): Exibe a opção para listar todos os usuários cadastrados.
  • print("4. Remover usuário"): Exibe a opção para remover um usuário com base no email.
  • print("5. Sair"): Exibe a opção para sair do aplicativo.
  • escolha = input("Escolha uma opção: "): Solicita ao usuário que escolha uma opção do menu.

                if escolha == '1':

  • idade = input("Digite a idade do usuário (opcional): "): Solicita a idade do usuário (opcional).
  • bio = input("Digite a bio do usuário (opcional): "): Solicita a bio do usuário (opcional).
  • idade = int(idade) if idade else None: Converte a idade para um inteiro se fornecida; caso contrário, define como None.
  • cadastro.adicionar_usuario(nome, email, idade, bio): Chama o método adicionar_usuario para adicionar o usuário ao banco de dados.

  • for usuario in usuarios:: Itera sobre a lista de usuários e imprime cada um.

    • 5. Execução do Script

    
                        if __name__ == "__main__":

  • if __name__ == "__main__":: Garante que a função main só será chamada se o script for executado diretamente, e não se for importado como um módulo em outro script.

    Resumo

    Este script oferece uma interface de linha de comando para interagir com a classe CadastroUsuario. O usuário pode adicionar, consultar, listar e remover usuários do banco de dados. A função main controla o fluxo do programa, exibindo um menu de opções e processando a entrada do usuário em um loop até que o usuário escolha sair.
